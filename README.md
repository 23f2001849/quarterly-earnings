# Q4 2024 Quarterly Earnings Report

## Interactive RevealJS Presentation

This repository contains an interactive quarterly earnings report presentation built with RevealJS, showcasing modern web presentation capabilities for financial reporting.

### Features

- ✅ **Interactive RevealJS slides** with smooth transitions
- ✅ **Email included**: 23f2001849@ds.study.iitm.ac.in
- ✅ **Markdown slide**: Market Position & Competitive Landscape section
- ✅ **Animated elements**: Fragment animations and highlight effects
- ✅ **Code samples**: Python financial analysis with syntax highlighting
- ✅ **Mathematical equations**: Financial formulas using MathJax
- ✅ **Speaker notes**: Comprehensive presentation guidance
- ✅ **Professional design**: Dark theme with financial data visualization

### Presentation Sections

1. **Title Slide** - Introduction with contact information
2. **Executive Summary** - Key financial metrics with animated cards
3. **Revenue Performance** - Interactive charts and growth analysis
4. **Profitability Analysis** - Margin expansion and key drivers
5. **Key Financial Metrics** - Mathematical formulas and ratios
6. **Code Sample** - Python financial analysis demonstration
7. **Market Position** - Markdown-formatted competitive analysis
8. **Risk Assessment** - Risk factors and mitigation strategies
9. **Q1 2025 Outlook** - Forward-looking guidance and priorities
10. **Thank You** - Closing slide with contact information

### Technical Implementation

- **RevealJS 4.3.1** - Presentation framework
- **ECharts.js** - Interactive data visualization
- **MathJax 3.2.0** - Mathematical equation rendering
- **Highlight.js** - Code syntax highlighting
- **Font Awesome** - Professional icons

### Usage

1. Open `index.html` in a modern web browser
2. Use arrow keys or navigation controls to move between slides
3. Press 'S' to open speaker notes (presenter view)
4. Press 'F' or 'ESC' to enter/exit fullscreen mode
